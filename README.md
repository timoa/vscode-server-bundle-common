# VSCode Server Templates

Docker images and Docker Compose that provides an easier way to install all the tooling on a VSCode Server instance.

## Common template

Includes:

* Pre-commit
* Ansible
* OpenJDK JDK
* OpenJDK JRE
* NodeJS (via nvm)
* Semantic Release


## DevOps template

Includes:

* Terraform
* Terraform Docs
* TFLint
* TFSec
* Checkov
* Kubectl
* Helm

## JS template

Includes:

* ESLint
* Prettier
* Yarn
